# Sakua HP 画像生成ブリーフ（Codex / image-2用）

## 共通スタイルルール（全アセット共通で指示に含める）

```
スタイル共通ルール:
- フラットデザイン。光沢・グロー効果・グラデーション禁止
- 色数は最大3色: ダークネイビー(#111827) + ホワイト(#ffffff) + アクセントブルー(#2563eb)
- マットな質感。少しだけ紙のようなノイズテクスチャを加える
- 完全な左右対称にしない。少し非対称な配置
- AIが生成した感じを消す。過剰にクリーン・過剰にシャープにしない
- 日本語テキストを含む場合はNoto Sans JP Boldを想定した太めのゴシック体
- 写真的なフォトリアルは避ける。グラフィックデザインとして作る
```
